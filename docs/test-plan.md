# Test Plan

- 验证目的地数量、设施种类和边数量达到课程要求
- 验证 Top-K 推荐不依赖全量排序，并覆盖同分场景稳定性
- 验证 Quickselect 对评分函数调用次数为线性规模（避免重复计算）
- 验证精确查询、前缀查询和多关键字联合查询正确
- 验证单点最短路、单源最短路距离表与路径代价一致
- 验证多点闭环在小规模使用 Held-Karp，大规模使用 Nearest Neighbor + 2-opt
- 验证多点闭环空目标输入可稳定返回，不触发异常
- 验证附近设施按图距离排序，并通过单源最短路复用降低查询开销
- 验证室内导航支持跨层路径规划，并可在轮椅模式下规避楼梯边
- 验证日记全文检索走索引路径，并可正确返回详情
- 验证日记浏览计数与登录评分互动链路（多用户可写）
- 验证 Huffman 压缩回环及压缩率指标输出
- 验证日记 AIGC 动画接口可生成分镜脚本、旁白串联和总时长
