# User Guide

## 启动后端

1. 进入 `backend/`
2. 激活 `backend/.venv`
3. 安装依赖并运行 `uvicorn app.main:app --reload`

## 启动前端

1. 进入 `frontend/`
2. 执行 `npm install`
3. 执行 `npm run dev`

## 演示流程

1. 查看首页概览
2. 打开目的地推荐页
3. 在搜索页查询校园或景区
4. 在地图导航页规划路线
5. 在地图导航页使用室内导航（大门到电梯、跨层到房间）
6. 查询附近设施和美食
7. 浏览与检索旅游日记，演示浏览计数与登录评分
8. 演示日记压缩与解压回放
9. 演示日记 AIGC 动画分镜生成与播放预览
